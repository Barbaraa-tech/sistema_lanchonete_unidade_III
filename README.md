# Sistema de Lanchonete

## Disciplina
Laboratório de Algoritmos - PET2018 - T01 (2026.1)

## Descrição do Projeto

Este projeto consiste no desenvolvimento de um sistema de lanchonete em linguagem C.

O sistema permite realizar o gerenciamento do cardápio através das operações básicas de CRUD (Criar, Listar, Atualizar e Remover produtos), utilizando vetores estáticos, estruturas de dados, funções, estruturas condicionais e de repetição.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Atualização de produtos
- Remoção de produtos
- Validação de dados
- Controle do limite máximo de produtos

## Como compilar

Utilizando o GCC:

```bash
gcc main.c produtos.c -o lanchonete
```

## Como executar

Após compilar:

```bash
./lanchonete
```
